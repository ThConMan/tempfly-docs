Documentation

Everything TempFly does, in plain English. Built for Paper / Minecraft 26.x, Java 21+.

Overview

TempFly gives players temporary flight that drains only while they're airborne — standing on the ground pauses the timer. Time is granted through Fly Vouchers: signed feather items players right-click to top up their balance. Balances live in SQLite and survive restarts, and a fall-grace window keeps players safe when their time runs out.

Because the timer only ticks in the air, players never "waste" flight time sorting chests or building on the ground. They pay for the part they actually use.

Install

  1. Download TempFly-x.y.z.jar.
  2. Drop it in your server's plugins/ folder (remove any older TempFly jar).
  3. Restart the server.
  4. Adjust plugins/TempFly/config.yml and run /tempfly reload. That's it.

Optional soft-depend (auto-detected): WorldGuard, for the temp-fly region flag. SQLite storage is built in — no database server needed.

How flight drains

Toggle flight with /tempfly. While you're in the air, your banked time counts down; the moment you land or stand still, the timer pauses. A live action-bar countdown shows the remaining time and shifts colour as it runs low:

  • Green — plenty of time left.
  • Yellow — running low; think about landing.
  • Red — almost out; flight is about to end.

When the balance hits zero, flight switches off and the fall-grace window kicks in so you can glide down safely.

Fly Vouchers

Fly Vouchers are the way time enters the economy. A voucher is a feather item (the material and name/lore are configurable) that grants a set amount of flight time when right-clicked.

Right-click to redeem

The player right-clicks the voucher; the time is added to their balance and the item is consumed.

HMAC-signed

Every voucher carries an HMAC signature, so vouchers can't be forged, hand-crafted, or duplicated to mint free flight time.

Give from console

Use /tempfly give <player> <duration> [count] — wire it into crates, shops, or rewards.

Stack & trade

Vouchers are normal items players can carry, store, and trade until they redeem them.

Persistence & safety

SQLite storage

Flight balances are stored in a built-in SQLite database and survive restarts and crashes.

Mid-air restore

Log off while flying and your flight is restored when you log back in, so a disconnect never strands you.

Fall grace

When flight ends, a few seconds of fall-damage immunity let you descend safely instead of taking the full drop.

Commands

Base command /tempfly — alias /tfly. Admin subcommands require tempfly.admin.

CommandWhat it does
/tempflyToggle your own temporary flight on or off.
/tempfly timeCheck your remaining flight balance.
/tempfly give <player> <duration> [count]Give a player [count] Fly Vouchers worth <duration> each. admin
/tempfly add <player> <duration>Add flight time directly to a player's balance. admin
/tempfly set <player> <duration>Set a player's balance to an exact amount. admin
/tempfly check <player>Check another player's balance. admin
/tempfly reloadReload the config. admin

Config

The whole config.yml is commented in plain language. The key knobs:

voucher:
  material: FEATHER       # the item used for Fly Vouchers
  name: "&b&lFly Voucher"
  lore:
    - "&7Right-click to bank flight time."

drain:
  airborne-only: true     # time only counts down while flying
  actionbar: true         # show the live green->yellow->red countdown

fall-grace:
  enabled: true
  seconds: 4              # fall-damage immunity when flight ends

storage:
  type: sqlite            # built-in, file-based; survives restarts

worldguard:
  enabled: true           # register the temp-fly region flag

WorldGuard

If WorldGuard is installed, TempFly registers a temp-fly region flag. Set it to deny on a region to forbid temporary flight there:

/rg flag spawn temp-fly deny

Players keep their banked time — they simply can't activate temp flight inside that region. Anywhere the flag isn't denied, flight works as normal.

Use this to keep arenas, spawn, and event areas grounded without touching anyone's balance.

FAQ

Does time drain while I'm standing still?

No. With airborne-only on (the default), the timer only ticks while you're actually in the air.

Can players dupe vouchers?

No. Vouchers are HMAC-signed, so forged or hand-crafted copies are rejected.

What happens if I log off mid-flight?

Your balance is saved and your flight is restored when you log back in.

Do I need a database server?

No. Storage is built-in SQLite — a single file, no setup.

Will I take fall damage when time runs out?

No. A short fall-grace window lets you land safely.

Do I need WorldGuard?

No — it's an optional hook. Without it, flight simply works everywhere.
